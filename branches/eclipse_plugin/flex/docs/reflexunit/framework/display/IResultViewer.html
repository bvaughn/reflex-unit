Packagereflexunit.framework.display
Interfacepublic interface IResultViewer extends ITestWatcher
ImplementorsConsoleViewer, CruiseControlLogger

An IResultViewer relates the status of a test as it is executed. This is often done using a graphical interface but may also be done using plain text, XML, sockets, etc.
