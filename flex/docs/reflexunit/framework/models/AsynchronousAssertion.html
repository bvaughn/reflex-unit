Classpublic class AsynchronousAssertion
InheritanceAsynchronousAssertion flash.events.EventDispatcher

Wrapper class used to contain a single test method call to TestCase.addAsync.

This class will dispatch an Event.COMPLETE event to indicate that the asynchronous assertion completed successfully.

It will dispatch and ErrorEvent.ERROR error to indicate that the assertion failed. Failure can be due to a runtime Error or an AssertFailedError. In either case, this value will be stored in error.
