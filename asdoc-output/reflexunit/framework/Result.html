Packagereflexunit.framework
Classpublic class Result

Used to collect and describe the results of running a test. In most cases, a Result object should only be used (ie. run) once and then discared.

The test framework distinguishes between failures and errors. A failure is anticipated and checked for with assertions. Errors are unanticipated problems like an RangeError.
