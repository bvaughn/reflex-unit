Classpublic class RunNotifier
InheritanceRunNotifier flash.events.EventDispatcher
ImplementsITestWatcher

A RunNotifier is used by several of the testing framework classes to monitor the status of executing tests. It is typically passed as a parameter to a run method and listened to by the caller class for events.
