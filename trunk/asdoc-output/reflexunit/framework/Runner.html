Packagereflexunit.framework
Classpublic class Runner

This class is responsible for the high-level management of test execution. That includes the following actions:
  1. Convert a TestSuite, if provided, into a Recipe of testable methods
  2. Execute each of those methods and store their outcome in a Result object
  3. Inform a given set of IResultViewer objects of its progress by way of a RunNotifier method

public static function create(testSuiteOrRecipeIn:*, resultViewers:Array, runNotifier:RunNotifier = null, resultIn:Result = null):Result

Convenience method for creating and executing a Runner instance. This method will automatically run the newly created Runner and return its Result object.

Parameters
testSuiteOrRecipeIn:* — TestSuite or Recipe containing tests to be run
 
resultViewers:Array — Array containing IResultViewer instances to be notified of test progress
 
runNotifier:RunNotifier (default = null) — (Optional) RunNotifier to use if caller wishes to be informed of progress
 
resultIn:Result (default = null) — (Optional) Result to store progress in; if no value is supplied a new Result will be created
